# Story: E3-S3 - Implement Audio Recording Logic

## Story Info
- **Epic:** E3 - Voice Recording
- **Sprint:** 2
- **Priority:** Must Have
- **Points:** 3
- **Status:** not_started
- **Dependencies:** E3-S1, E3-S2

## Description

As a user, I need the app to capture audio when I record so that my voice note can be transcribed.

## Acceptance Criteria

- [ ] Audio recording starts on record screen mount (after permission check)
- [ ] Recording uses `expo-av` `Audio.Recording` API
- [ ] Audio captured in m4a format with 16kHz+ sample rate (Whisper-compatible)
- [ ] Audio metering data fed to `AudioWaveform` component during recording
- [ ] Tapping Stop ends recording and returns local audio file URI
- [ ] 5-minute (300 second) maximum recording duration enforced
- [ ] Auto-stop at 5 minutes with user notification (alert or toast)
- [ ] Recording produces a valid audio file that can be played back
- [ ] Audio mode configured correctly (allowsRecordingIOS, playsInSilentModeIOS)

## Technical Notes

### Recording Configuration
```typescript
import { Audio } from 'expo-av';

const RECORDING_OPTIONS: Audio.RecordingOptions = {
  isMeteringEnabled: true,
  android: {
    extension: '.m4a',
    outputFormat: Audio.AndroidOutputFormat.MPEG_4,
    audioEncoder: Audio.AndroidAudioEncoder.AAC,
    sampleRate: 44100,
    numberOfChannels: 1,
    bitRate: 128000,
  },
  ios: {
    extension: '.m4a',
    outputFormat: Audio.IOSOutputFormat.MPEG4AAC,
    audioQuality: Audio.IOSAudioQuality.HIGH,
    sampleRate: 44100,
    numberOfChannels: 1,
    bitRate: 128000,
  },
  web: {
    mimeType: 'audio/webm',
    bitsPerSecond: 128000,
  },
};
```

### Recording Flow
```typescript
// Set audio mode
await Audio.setAudioModeAsync({
  allowsRecordingIOS: true,
  playsInSilentModeIOS: true,
});

// Create and start recording
const recording = new Audio.Recording();
await recording.prepareToRecordAsync(RECORDING_OPTIONS);
await recording.startAsync();

// Get metering data (poll status)
const status = await recording.getStatusAsync();
// status.metering is dB value for waveform

// Stop recording
await recording.stopAndUnloadAsync();
const uri = recording.getURI(); // local file path
```

### 5-Minute Limit
```typescript
const MAX_DURATION_MS = 5 * 60 * 1000; // 300000ms

// Check in status polling interval
if (status.durationMillis >= MAX_DURATION_MS) {
  await stopRecording();
  Alert.alert('Recording limit reached', 'Maximum recording duration is 5 minutes.');
}
```
