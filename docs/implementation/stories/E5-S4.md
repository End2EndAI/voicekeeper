# Story: E5-S4 - Implement Note Deletion

## Story Info
- **Epic:** E5 - Note Management
- **Sprint:** 3
- **Priority:** Must Have
- **Points:** 2
- **Status:** not_started
- **Dependencies:** E5-S3

## Description

As a user, I need to delete notes I no longer need so that my note collection stays clean and relevant.

## Acceptance Criteria

- [ ] Delete button (trash icon) on Note Detail screen header
- [ ] Tapping delete shows confirmation dialog: "Delete this note? This cannot be undone."
- [ ] Confirming delete calls `deleteNote` from NotesContext
- [ ] Note removed from database and local state
- [ ] Navigation returns to Home after deletion
- [ ] Home grid updates immediately (note no longer visible)
- [ ] Cancel in dialog keeps note and stays on detail screen
