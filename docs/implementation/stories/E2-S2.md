# Story: E2-S2 - Build Login/Signup Screen

## Story Info
- **Epic:** E2 - Authentication
- **Sprint:** 1
- **Priority:** Must Have
- **Points:** 3
- **Status:** not_started
- **Dependencies:** E2-S1

## Description

As a user, I need a login/signup screen so that I can create an account or sign in to access my notes.

## Acceptance Criteria

- [ ] Login screen at `app/login.tsx` with email and password fields
- [ ] Toggle between "Sign In" and "Sign Up" modes
- [ ] Sign Up mode calls `supabase.auth.signUp({ email, password })`
- [ ] Sign In mode calls `supabase.auth.signInWithPassword({ email, password })`
- [ ] Validation errors displayed clearly:
  - Empty email or password
  - Invalid email format
  - Password too short (Supabase minimum: 6 chars)
  - Wrong credentials on sign in
  - Email already registered on sign up
- [ ] Loading indicator shown during auth request
- [ ] Successful auth triggers navigation to Home screen (via AuthContext state change)
- [ ] App branding/logo displayed on login screen

## Technical Notes

### Screen Layout
- App name "VoiceKeeper" at top
- Email input (TextInput, keyboard type: email-address, autoCapitalize: none)
- Password input (TextInput, secureTextEntry)
- Primary action button (Sign In or Sign Up)
- Toggle link: "Don't have an account? Sign Up" / "Already have an account? Sign In"
- Error message area below form

### Error Handling
```typescript
const { error } = await supabase.auth.signInWithPassword({ email, password });
if (error) {
  setErrorMessage(error.message);
}
```

## Files to Create/Modify
- `app/login.tsx` - Login/signup screen

## Testing
