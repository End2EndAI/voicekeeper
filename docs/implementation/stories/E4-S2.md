## Description

As a user, I need my transcription to be auto-formatted into my chosen format so that my notes are structured and organized without manual editing.

## Acceptance Criteria

- [ ] After Whisper transcription, gpt-5-nano is called for formatting
- [ ] Format-specific prompts implemented for all 4 types:
  - `bullet_list`: Key points as bulleted list
  - `paragraph`: Clean, readable prose
  - `action_items`: Tasks as checkbox items (`- [ ] task`)
  - `meeting_notes`: Structured with Key Topics, Decisions, Action Items sections
- [ ] Title auto-generated (5 words max) from the transcription content
- [ ] GPT returns structured JSON: `{ title, content }`
- [ ] Response returns complete result: `{ transcription, formatted_text, title }`
- [ ] GPT errors fall back gracefully:
  - `formatted_text` = raw transcription
  - `title` = first 5 words of transcription
- [ ] All format types produce high-quality, usable output

## Technical Notes

### GPT Formatting Call
```typescript
const formatTranscription = async (
  transcription: string,
  formatType: string
): Promise<{ title: string; content: string }> => {
  const systemPrompts: Record<string, string> = {
    bullet_list: 'Extract the key points from this transcription and format them as a concise bulleted list using markdown bullets (- ).',
    paragraph: 'Rewrite this transcription as clean, well-structured prose paragraphs. Fix grammar, remove filler words, and improve readability.',
    action_items: 'Extract all action items and tasks from this transcription. Format each as a checkbox item using markdown (- [ ] task description).',
    meeting_notes: 'Structure this transcription as meeting notes with the following markdown sections: ## Key Topics, ## Decisions Made, ## Action Items. Use bullet points within each section.',
  };

  const response = await fetch('https://api.openai.com/v1/chat/completions', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${Deno.env.get('OPENAI_API_KEY')}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      model: 'gpt-5-nano',
      messages: [
        {
          role: 'system',
          content: `${systemPrompts[formatType]}\n\nAlso generate a short title (5 words maximum) summarizing the content.\n\nRespond with valid JSON only: {"title": "...", "content": "..."}`
        },
        { role: 'user', content: transcription }
      ],
      temperature: 0.3,
      response_format: { type: 'json_object' },
    }),
  });

  const result = await response.json();
  return JSON.parse(result.choices[0].message.content);
};
```

### Fallback on Error
```typescript
try {
  const formatted = await formatTranscription(transcription, formatType);
  return { transcription, formatted_text: formatted.content, title: formatted.title };
} catch (error) {
  // Fallback: use raw transcription
  const words = transcription.split(' ');
  const fallbackTitle = words.slice(0, 5).join(' ');
  return { transcription, formatted_text: transcription, title: fallbackTitle };
}
```
