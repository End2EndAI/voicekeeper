# Story: E5-S3 - Build Note Detail Screen

## Story Info
- **Epic:** E5 - Note Management
- **Sprint:** 3
- **Priority:** Must Have
- **Points:** 3
- **Status:** not_started
- **Dependencies:** E5-S2

## Description

As a user, I need to view and edit a full note so that I can read the complete content and make corrections.

## Acceptance Criteria

- [ ] Note detail screen at `app/note/[id].tsx`
- [ ] Loads note data by id from NotesContext
- [ ] Displays:
  - Title (editable)
  - Full formatted text (editable)
  - Format type badge
  - Created date
  - Last updated date
- [ ] Edit mode: tap text area to start editing (multiline TextInput)
- [ ] Save button appears when changes are made
- [ ] Save calls `updateNote` and shows success feedback
- [ ] Back navigation returns to Home
- [ ] Handle case where note id is not found (navigate back with error)

### Layout
- Header with back button and delete icon (delete wired in E5-S4)
- Title area (large font, tappable to edit)
- Format badge + timestamps row
- Scrollable text content area
- Sticky Save button at bottom when changes exist
