# Story: E6-S2 - Build Settings Screen

## Story Info
- **Epic:** E6 - Settings and Preferences
- **Sprint:** 3
- **Priority:** Must Have
- **Points:** 2
- **Status:** not_started
- **Dependencies:** E6-S1

## Description

As a user, I need a Settings screen where I can choose my default note format and log out.

## Acceptance Criteria

- [ ] Settings screen at `app/settings.tsx`
- [ ] Accessible from Home screen header (gear icon)
- [ ] Displays 4 format options: Bullet List, Paragraph, Action Items, Meeting Notes
- [ ] Currently selected format is highlighted (checkmark or filled radio)
- [ ] Tapping a format option updates the preference immediately via PreferencesContext
- [ ] Selection persists to Supabase (verified by closing/reopening app)
- [ ] Logout button at bottom of screen
- [ ] Logout clears session and navigates to Login (via AuthContext signOut)
- [ ] Move any temporary logout button from Home header to this screen

## Technical Notes

### Format Selector
```typescript
const FORMAT_OPTIONS = [
  { value: 'bullet_list', label: 'Bullet List', description: 'Key points as bullets' },
  { value: 'paragraph', label: 'Paragraph', description: 'Clean prose formatting' },
  { value: 'action_items', label: 'Action Items', description: 'Tasks with checkboxes' },
  { value: 'meeting_notes', label: 'Meeting Notes', description: 'Structured meeting format' },
];

// Render as selectable list items
{FORMAT_OPTIONS.map(option => (
  <Pressable key={option.value} onPress={() => setDefaultFormat(option.value)}>
    <Text style={[styles.label, selected && styles.selectedLabel]}>
      {option.label}
    </Text>
    <Text style={styles.description}>{option.description}</Text>
    {selected && <CheckIcon />}
  </Pressable>
))}
```

### Layout
- Header: "Settings" title with back button
- Section: "Default Format" with format options list
- Section: "Account" with logout button
- Back navigation returns to Home
