# Story: E3-S4 - Implement Cancel Recording

## Story Info
- **Epic:** E3 - Voice Recording
- **Sprint:** 2
- **Priority:** Should Have
- **Points:** 1
- **Status:** not_started
- **Dependencies:** E3-S3

## Description

As a user, I need to be able to cancel a recording in progress so that I can discard unwanted recordings without saving.

## Acceptance Criteria

- [ ] Cancel button on recording screen discards the recording
- [ ] Recording is stopped and unloaded without saving the audio file
- [ ] User navigates back to Home screen after cancel
- [ ] If recording is longer than 10 seconds, show confirmation dialog before canceling
- [ ] If recording is 10 seconds or less, cancel immediately without confirmation

## Technical Notes

### Cancel Flow
```typescript
const handleCancel = async () => {
  if (elapsedSeconds > 10) {
    Alert.alert(
      'Discard Recording?',
      'Your recording will be lost.',
      [
        { text: 'Keep Recording', style: 'cancel' },
        { text: 'Discard', style: 'destructive', onPress: discardAndGoBack },
      ]
    );
  } else {
    await discardAndGoBack();
  }
};

const discardAndGoBack = async () => {
  await recording.stopAndUnloadAsync();
  router.back();
};
```
