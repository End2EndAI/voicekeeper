# Story: E5-S2 - Build Home Screen with Note Grid

## Story Info
- **Epic:** E5 - Note Management
- **Sprint:** 3
- **Priority:** Must Have
- **Points:** 5
- **Status:** not_started
- **Dependencies:** E5-S1

## Description

As a user, I need a Home screen with a card grid of my notes so that I can browse and access my voice notes quickly.

## Acceptance Criteria

- [ ] Home screen (`app/index.tsx`) displays a 2-column card grid of notes
- [ ] `NoteGrid` component renders notes using FlatList with numColumns=2
- [ ] `NoteCard` component displays:
  - Auto-generated title (bold, truncated to 1 line)
  - Text preview (truncated to 3 lines)
  - Timestamp (relative or formatted date)
  - `FormatBadge` showing format type (e.g., "Bullets", "Meeting")
- [ ] Tapping a NoteCard navigates to note detail: `router.push(/note/${id})`
- [ ] `RecordButton` FAB positioned at bottom-center, always visible
- [ ] Settings gear icon in header (right side)
- [ ] Notes sorted by creation date, newest first
- [ ] Empty state when no notes: friendly message + prompt to record first note
- [ ] Pull-to-refresh to reload notes
- [ ] Notes load on screen mount via NotesContext
