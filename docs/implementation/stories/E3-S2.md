## Description

As a user, I need a focused recording screen with visual feedback so that I know the app is capturing my voice.

## Acceptance Criteria

- [ ] Recording screen (`app/record.tsx`) opens as a modal from Home
- [ ] Prominent Stop button (large, centered, red/accent color)
- [ ] Cancel button (secondary, top-left or alongside Stop)
- [ ] Elapsed time display (MM:SS format, updates every second)
- [ ] `AudioWaveform` component shows animated audio level visualization
- [ ] Layout is focused and distraction-free (dark or minimal background)
- [ ] Screen fills available space (modal presentation)

### Timer
```typescript
// Simple elapsed time using useEffect + setInterval
const [elapsed, setElapsed] = useState(0);
useEffect(() => {
  if (!isRecording) return;
  const interval = setInterval(() => setElapsed(e => e + 1), 1000);
  return () => clearInterval(interval);
}, [isRecording]);

const formatTime = (seconds: number) => {
  const m = Math.floor(seconds / 60).toString().padStart(2, '0');
  const s = (seconds % 60).toString().padStart(2, '0');
  return `${m}:${s}`;
};
```
